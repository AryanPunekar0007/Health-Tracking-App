# Nutrition & Health Tracking App

## Overview
An Android mobile app developed in Kotlin with Jetpack Compose that enables users to track daily nutrition and health metrics.

## Features
- Intuitive UI built with Jetpack Compose components (forms, buttons, date pickers, progress bars)
- Real-time input validation for accurate data entry
- Local food database handling via CSV parsing
- Persistent user data storage with SharedPreferences

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## How to Run 
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run on an emulator or Android device.
